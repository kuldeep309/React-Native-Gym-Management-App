# React-Native-Gym-Management-App

A React Native gym management system prototype.

## Sprint 1 Completed

- Welcome / onboarding screen
- User registration
- User login
- Input validation
- Local user data storage using AsyncStorage
- Navigation between screens
- Enhanced UI design

## Sprint 2 Completed

- Membership plans (Basic, Standard, Premium)
- Plan selection with interactive UI
- Selected plan stored using AsyncStorage
- Billing details displayed (Plan, Price, Status, Billing Date)
- Home screen updated with membership information
- Improved UI design with scroll support
- Fixed navigation flow (Register → Login → Home → Membership → Home)

## Sprint 3 Completed

- Class schedule screen UI
- Display available gym classes
- Class selection and booking functionality
- Store selected class using AsyncStorage
- Display selected class on Home screen
- Navigation between Home and Classes screen
